# Натяжение — лендинг натяжных потолков (Москва и МО)

Продающий одностраничник для компании «Натяжение»: натяжные потолки под ключ с фиксированной ценой в договоре и гарантией 15 лет.

**Стек:** статический сайт (один `index.html`), Tailwind (CDN), vanilla JS. Без билд-шага — готов для GitHub Pages.

## Возможности
- Hero с фоновым видео (день → вечер, подсветка)
- Квиз-калькулятор стоимости (5 шагов) с расчётом скидки 10%
- Счётчик акции до конца недели, поп-ап на выход, липкая мобильная CTA
- WhatsApp-интеграция, плейсхолдеры под Яндекс.Метрику и цели

## Структура
- `index.html` — весь сайт
- `assets/web/*.jpg` — оптимизированные изображения
- `assets/hero-video.mp4` — hero-видео

> Данные с пометкой ⚠️ (опыт, кол-во работ, цены, отзывы, реквизиты) — демонстрационные, заменяются на реальные данные клиента.
